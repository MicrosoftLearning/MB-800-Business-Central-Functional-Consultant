## Exercise 3: Perform actions with Copilot


## Task 1: Autofill fields with Copilot

## Scenario
You are working as a Purchaser at Contoso, and the company has decided to expand its product offering by selling Microsoft devices and software. The first sales order has already been received, but before you can process it, you must set up the required master data in Business Central.

To support this new product line, you need to create a vendor card for Microsoft and retail item cards for the products included in the sales order:

- Surface Laptop 7

- Copilot Pro

Contoso has recently enabled Copilot features in Business Central, allowing you to streamline data entry.

You will perform:
1. Set up Vendor card using autofill fields with Copilot.
2. Set up Item cards using autofill fields with Copilot.

### Steps
1. Set up Vendor card using autofill fields with Copilot.
	Select the Search icon in the top-right corner of the page, enter Vendors, and then choose the related link.

In the Vendors page, select +New.
	Select DOMESTIC on the Select Template for a new vendor page.
	Select OK.
	On the new vendor card enter Microsoft in the Name field and
	Select the icon in the right corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page expand and check the list of Copilot Suggestion
	Select Keep Selected.
	On the Address&Contact FastTab select the US in the field Country/Region Code.
	In the field Home Page 
	Select the icon in the right corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page expand and check the list of Copilot Suggestion
	Select Keep Selected.
	Close Vendor card.

2. Set up Item cards using autofill fields with Copilot.
	Select the Search icon in the top-right corner of the page, enter Items, and then choose the related link.
	In the Items page, select +New.
	Select COMPHW on the Select Template for a new item page.
	Select OK.
	On the new item card enter Surface Laptop 7 in the Name field and
	Select the icon in the right corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page, expand and check the list of Copilot Suggestion
	Select Keep Selected.
	On the Prices&Sales FastTab select the field Unit Price.
	Select the icon in the left corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page, expand and check the list of Copilot Suggestion.
	Select Keep Selected.
	In the Items page, select +New.
	Select SUBSCRIPTION on the Select Template for a new item page.
	Select OK.
	On the new item card enter Copilot Pro in the Name field and
	Select the icon in the right corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page, expand and check the list of Copilot Suggestion
	Select Keep Selected.
	On the Prices&Sales FastTab select the field Unit Price.
	Select the icon in the left corner of the field Suggest value for this field and related fields on this FastTab.
	In the top-right corner of the page, expand and check the list of Copilot Suggestion.
	Select Keep Selected.

## Task 2. Marketing text suggestions with Copilot

## Scenario
Contoso has recently expanded its product catalog and will begin selling Microsoft products through its new Shopify storefront. To prepare for the launch, the marketing and sales teams need compelling product descriptions that can be published directly to Shopify. Instead of writing these descriptions manually, Contoso wants to leverage Copilot in Business Central to generate highquality marketing text quickly and consistently.

As the purchaser responsible for onboarding new products, you must prepare the item cards for Surface Laptop 7.

### Steps
1. Select the Search icon in the top-right corner of the page, enter Items, and then choose the related link.
2. Select the item for Surface Laptop 7, select Edit.
3. Select Item, Attributes.
4. Select the Color, +New, enter Platinum.
5. Enter Snapdragon X Elite in the Processor.
6. Select OK.
7. Select in the right corner the icon ! Expand the FactBox pane.
8. Scroll to the Marketing text section.
9. Select the Create Draft.
10. Read the draft, make some corrections.
11. Select Keep It.

## Task 3. Suggest lines on sales documents with Copilot.

## Scenario
You are working as a Sales Coordinator at Contoso, and the company has just received a new sales order from Customer 10000 – Adatum Corporation. The order includes several Microsoft products that Contoso recently added to its catalog: Surface Laptop 7 and Copilot Pro. Instead of manually entering each sales line, you can now use Copilot in Business Central to streamline the process.

### Steps
1. Select the Search icon in the top-right corner of the page, enter Sales Orders, and then choose +New.
2. In the No. field, select 10000 (Adatum Corporation).
3. In the Lines section, select the Copilot icon,
4. Select Suggest Sales Lines.
5. Enter Surface and Copilot
6. Select Generate.
7. Select all lines and select Insert.
